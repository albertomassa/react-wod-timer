
# react-wod-timer

A simple wod timer created with React

## Getting Started

### Project Run

```
yarn install
yarn start
localhost:3000
```

### Project Build

```
yarn build
```

### Docker

```
docker build -t react-wod-timer .
docker run -p 80:80 react-wod-timer
```
